Essential HTML Tags

Tag Purpose Example
<h1> - <h6> Headings <h1>Main Title</h1>
<p> Paragraph <p>Text</p>
<a> Link <a href="url">Click</a>
<img> Image <img src="img.jpg" alt="desc">
<ul> / <ol> Lists <ul><li>Item</li></ul>
<div> Block container <div>Content</div>

HTML Attributes

Attributes add extra information to tags. They're always written inside the opening tag.

  • href: Sets the link in <a>
  • src: Sets the image source in <img>
  • alt: Adds alternative text for images
  • class / id: Used for CSS and JavaScript
  • title: Tooltip on hover

Example:

<a href="https://example.com" title="Visit site">Visit Example</a>

Basic HTML Page Structure

A typical HTML5 page includes semantic elements for better structure and readability.

<!DOCTYPE html>
<html>
  <head>
    <title>Page Title</title>
  </head>
  <body>
    <header>Header content</header>
    <nav>Navigation</nav>
    <main>
      <section>Main content</section>
    </main>
    <footer>Footer info</footer>
  </body>
</html>
