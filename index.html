About This Project

This site was created as part of my coding course to support my learning process and help others who are just starting out.

As I work through HTML, CSS, and JavaScript, I’ve found it incredibly helpful to have a quick and simple reference I can return to without having to dig through pages of documentation or videos.

This cheat sheet captures key information in a clear, organized way to make learning and revision easier.

Cheat sheets are especially useful because they provide instant reminders of syntax, structure, and common code patterns.

Instead of getting stuck trying to remember the correct tag or selector, you can simply glance at the reference and get back to building.

Whether you're practicing for an exam, working on a project, or revisiting the basics, having a personal coding cheat sheet saves time, reduces frustration, and boosts confidence.
